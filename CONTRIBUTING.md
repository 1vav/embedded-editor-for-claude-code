# Contributing

Thanks for your interest in contributing!

## Development Setup

```sh
git clone https://github.com/1vav/embedded-editor-for-claude-code.git
cd embedded-editor-for-claude-code
npm install
npm run build          # generate vendor/ bundles (required before running)
node bin/cli.js view   # start the viewer at http://127.0.0.1:3000
```

## Project Structure

```
bin/cli.js                  CLI entry point (init / serve / view)
src/server.js               MCP server — registers all tools
src/viewer-server.js        HTTP + SSE server — serves the browser UI and REST API
src/viewer/entry.jsx        Browser SPA (React + Excalidraw + tldraw + Markdown)
src/workspace.js            Shared file utilities (path validation, link rewriting)
src/render.js               PNG rendering pipeline (Excalidraw SVG → resvg → base64)
src/paths.js                ROOT resolution and path helpers
src/init.js                 `init` command — writes MCP config and CLAUDE.md
scripts/
  build-excalidraw-bundle.mjs   Pre-bundles @excalidraw/excalidraw for Node
  build-viewer-bundle.mjs       esbuild config for the React SPA
  smoke-stdio.mjs               MCP happy-path integration test
vendor/                     Pre-built bundles (generated by npm run build, gitignored)
```

## Build System

| Command | What it does | When to run |
|---------|-------------|-------------|
| `npm run build` | Runs both steps below | Before opening a PR |
| `npm run build:excalidraw` | Bundles `@excalidraw/excalidraw` for Node | After bumping Excalidraw version |
| `npm run build:viewer` | Bundles the React SPA | After editing `src/viewer/entry.jsx` |

The `vendor/` directory is gitignored but included in the npm package via the `files` field.

## Testing

```sh
node scripts/smoke-stdio.mjs
```

Verifies: MCP initialize handshake, all tools registered correctly, PNG rendering, file I/O.

## Making Changes

- **Branch:** create a feature branch off `main` (`feat/my-thing`, `fix/my-bug`)
- **Commits:** use [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `docs:`, `chore:`)
- **Before opening a PR:** run `npm run build` and `node scripts/smoke-stdio.mjs`
- **Vendor files:** if you changed `src/viewer/entry.jsx`, include the rebuilt `vendor/viewer.js` and `vendor/viewer.css` in your PR

## Security

Please do not open public issues for security vulnerabilities. Use [GitHub's private security advisory](https://github.com/1vav/embedded-editor-for-claude-code/security/advisories/new) instead. See [SECURITY.md](./SECURITY.md).
